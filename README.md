# Game Boy Advance SP AGS-001 Japanese Label

[中文](#中文说明) · [English](#english)

## 中文说明

用于日版 Game Boy Advance SP（AGS-001）背部外壳标签的高精度复刻模板，适合掌机换壳、旧标签修复和个人 DIY。

### Release 内容

项目文件通过 GitHub Release 提供：

- `AGS-001-JPN.psd`：可编辑的 Photoshop 分层源文件；
- `AGS-001-JPN.png`：可直接预览和打印的高分辨率导出图；
- `Assets.7z`：模板使用的配套素材和字体文件。

### 使用方法

1. 下载同一 Release 中的 PSD、PNG 和素材包。
2. 打开 PSD 前安装素材包中所需字体，避免字重和间距变化。
3. 如有需要，在 Photoshop 中修改条形码下方的序列号。
4. 按 `72.6 × 25.1 mm` 输出；推荐使用拉丝银或哑银 PET 不干胶，并覆盖哑光膜。

### 鸣谢与使用限制

模板中使用的 `OCR B Std` 字体素材来源记录为 [CharlieSIGMA/gb-label-creator](https://github.com/CharlieSIGMA/gb-label-creator)。感谢该项目对 Game Boy 标签字体的考据。

本项目用于掌机维修、翻新和个人 DIY，不应用于商业量产、冒充原厂配件或其他欺诈用途。仓库未提供涵盖全部素材的统一开源许可证；使用字体和第三方素材时请同时遵守其各自条款。

## English

High-accuracy replacement template for the Japanese Game Boy Advance SP (AGS-001) rear shell label, intended for restoration, reshelling, and personal DIY use.

The GitHub Release contains the editable `AGS-001-JPN.psd`, a ready-to-preview `AGS-001-JPN.png`, and an `Assets.7z` package with supporting assets and fonts. Install the required fonts before opening the PSD, edit the serial number if needed, and print at `72.6 × 25.1 mm`.

Brushed or matte silver PET sticker stock with a matte laminate is recommended. Do not use the template for counterfeiting or commercial mass production. Third-party fonts and assets retain their own terms.
